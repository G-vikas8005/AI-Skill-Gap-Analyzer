# AI Skill Gap Analyzer

AI-powered web application that analyzes resumes, evaluates ATS compatibility, identifies skill gaps, and provides personalized career guidance.

## Features

- Resume PDF Upload
- ATS Score Analysis
- Skill Gap Analysis
- AI-Powered Career Guidance
- Learning Roadmap Generation
- Interview Preparation Tips
- User Authentication (JWT)

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- React Router
- Recharts

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Multer
- PDF.js
- Gemini AI API
