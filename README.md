# AI Skill Gap Analyzer

AI Skill Gap Analyzer is a web application that analyzes resumes, checks ATS compatibility, identifies skill gaps for target job roles, and provides AI-powered career guidance.

## Features

* Upload and parse PDF resumes
* ATS score and resume analysis
* Skill gap detection
* Radar chart visualization
* AI-generated career guidance
* Learning roadmap and interview tips

## Tech Stack

### Frontend

* React.js
* Tailwind CSS
* Recharts
* Axios

### Backend

* Node.js
* Express.js
* MongoDB
* PDF.js

### AI

* Google Gemini API

## Supported Roles

* Frontend Developer
* Backend Developer
* Full Stack Developer
* Data Scientist

## Installation

### Backend

```bash
cd backend
npm install
npm run server
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Create a `.env` file in the backend:

```env
PORT=5000
MONGO_URI=your_mongodb_uri
GEMINI_API_KEY=your_gemini_api_key
```
